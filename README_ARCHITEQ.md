# Docker

### Run docker compose
```shell
 docker compose up --build
```

### Build and publish docker file
```shell
docker buildx build --platform linux/amd64 -t imarchiteq/sprintometer-ui-test:11.1.9 --load .
# or --platform linux/arm64

docker login

docker push imarchiteq/sprintometer-ui-test:11.1.9
```


# Database

### Postgres configuration example
```dockerfile
GF_DATABASE_TYPE=postgres
GF_DATABASE_HOST=postgres:5432
GF_DATABASE_NAME=grafana
GF_DATABASE_USER=grafana
GF_DATABASE_PASSWORD=grafana_password
```

### All database variables
```dockerfile
# Database type (mysql, postgres, sqlite3):
ENV GF_DATABASE_TYPE=postgres

# Database host:
ENV GF_DATABASE_HOST=127.0.0.1:3306



# Database name:
ENV GF_DATABASE_NAME=grafana

# Database username:
ENV GF_DATABASE_USER=root

# Database password:
ENV GF_DATABASE_PASSWORD=

# Database connection URL (if used):
ENV GF_DATABASE_URL=

# Maximum number of idle connections:
ENV GF_DATABASE_MAX_IDLE_CONN=2

# Maximum number of open connections:
ENV GF_DATABASE_MAX_OPEN_CONN=

# Maximum connection lifetime:
ENV GF_DATABASE_CONN_MAX_LIFETIME=14400

# SSL mode for the database (only for postgres and mysql):
ENV GF_DATABASE_SSL_MODE=disable

# Path to CA certificate:
ENV GF_DATABASE_CA_CERT_PATH=

# Path to client key:
ENV GF_DATABASE_CLIENT_KEY_PATH=

# Path to client certificate:
ENV GF_DATABASE_CLIENT_CERT_PATH=

# Server certificate name:
ENV GF_DATABASE_SERVER_CERT_NAME=

# Path to the database (only for sqlite3):
ENV GF_DATABASE_PATH=grafana.db

# Cache mode for sqlite3:
ENV GF_DATABASE_CACHE_MODE=private

# Enable WAL for sqlite3:
ENV GF_DATABASE_WAL=false

# Enable migration locking (only for mysql and postgres):
ENV GF_DATABASE_MIGRATION_LOCKING=true

# Migration lock timeout:
ENV GF_DATABASE_LOCKING_ATTEMPT_TIMEOUT_SEC=0

# Query retries for sqlite:
ENV GF_DATABASE_QUERY_RETRIES=0

# Transaction retries for sqlite:
ENV GF_DATABASE_TRANSACTION_RETRIES=5

# Enable metrics and tracing for database queries:
ENV GF_DATABASE_INSTRUMENT_QUERIES=false

```

# Plugins
### How to add core plugin?
Put source files to the public/app/plugins/panel/plugin-name directory.
Add import to the public/app/features/plugins/built_in_plugins.ts
